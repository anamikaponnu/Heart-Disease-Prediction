# Heart-Disease-Prediction
-This project focuses on predicting the likelihood of heart disease using machine learning techniques. The workflow includes data preprocessing, feature engineering, handling class imbalance, model training, evaluation, and visualization.
-The goal is to build and compare multiple models to identify the most effective approach for heart disease prediction.

Dataset Name: Heart Disease Cleveland Dataset
Source: UCI Machine Learning Repository
Link: https://archive.ics.uci.edu/ml/datasets/Heart+Disease
This dataset contains medical attributes such as age, sex, chest pain type, cholesterol levels, and more, used to predict the presence of heart disease.
